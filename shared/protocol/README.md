# Protocol

This directory defines the wire protocol used between AutoMatDeck Desktop
and Mobile.

## Contents

| File | Purpose |
|------|---------|
| `protocol-version.md` | Current version and changelog |
| `messages/` | Per-message schema definitions (future) |
| `schemas/` | JSON Schema or Protobuf definitions (future) |

## Stability

The protocol is currently **draft** (v0.x). Breaking changes are expected.

Once the first production release ships, the protocol will follow
[semantic versioning](https://semver.org/).
