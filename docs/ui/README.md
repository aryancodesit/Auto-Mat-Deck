# UI Documentation

> Wireframes, mockups, user flows, and design guidelines for the mobile application.

## Contents

- `wireframes/` — Screen mockups (added when available)
- `flows/` — User journey diagrams (added when available)
- `components/` — Reusable UI component specifications (added when available)

## Principles

- Mobile-first: all UI is designed for Android first.
- Desktop agent has no UI — it is configured from mobile.
- Touch-optimized, gesture-driven interactions.
- Dark and light theme support from day one.
