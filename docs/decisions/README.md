# Architectural Decisions

> This directory records Architectural Decision Records (ADRs) for the project.

## What is an ADR?

An Architectural Decision Record captures a significant decision, the context behind it, the options considered, and the rationale for the chosen approach.

[What prompted this decision?]

[What was decided?]

[Why was this chosen over alternatives?]
